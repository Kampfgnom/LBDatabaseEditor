void LBDatabase::RelationValueRight::fetchValue ( ) [private]

Fetches the value of this relation value.

If the cardinality is N:M it calls Relation::initializeManyToManyRelation(), which populates both the left and right values.

If the cardinality is 1:n or 1:1 this RelationValue right fetches its content and registers with the left value.
