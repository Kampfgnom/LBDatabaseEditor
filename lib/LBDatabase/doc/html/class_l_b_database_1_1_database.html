The Database class represents a low-level SQLite database.

Database * LBDatabase::Database::instance ( const QString &  fileName) [static]

Returns a Database instance, which works on the SQLite database fileName.

Since there may be only one instance per file, this method returns the same instance each time you call it with the same file name. This is useful because you can globally access the same instance of a database.

Note that you need to call open() the first time you instantiate a Database.
