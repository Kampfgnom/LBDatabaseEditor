Detailed Description

The Entity class models a concrete instance of an EntityType.

While EntityType represents a description for a group of entities, th Entity class is responsible for concrete single instances of such a type.

Each entity has a set of PropertyValues, one value for each Property of the corresponding EntityType. You may access these values one by one via propertyValue or as a list propertyValues().

You can create new entity instances with Context::insertEntity().
