Detailed Description

The Row class represents a row in a Table.

There exists an instance of Row for each row in each table. You can use data() and setData() to query an manipulate the content of fields.

You will seldomly instantiate this class on your own. Use the access and creation methods of Table instead (i.e. Table::appendRow() and Table::row()).
